# Ornate Mandir Wall Unit (Design 6)

A beautiful, wall-mounted mandir solution that saves floor space while providing a stunning focus for spiritual practice. This design features intricate details and a layout optimized for both murtis and wall-hung frames.

## Features
- **Space-Saving:** Securely mounts to the wall.
- **Intricate Patterns:** Subtle detailing on the side panels.
- **Integrated Lighting:** Soft glow to illuminate the sacred space.
- **Natural Texture:** Hand-selected birch for a premium organic feel.

## The Process
Focusing on wall-mounting safety and visual lightness, this mandir was engineered to be sturdy yet appear to float effortlessly.
