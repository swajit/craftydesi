# Scout Project 1

A collaborative build focused on community service and skill development. This project represents our commitment to mentoring and sharing the joy of woodworking with the next generation.

## Features
- **Community-Focused Design:** Built to meet a specific local need.
- **Robust Utility:** Designed for heavy use in public or shared spaces.
- **Educational Value:** Demonstrates fundamental woodworking principles.
- **Sustainable Materials:** Built using responsibly sourced timber.

## The Process
Working with the Scouts was a highlight. We focused on teaching safe tool use and the importance of planning and precision in every step of the build.
