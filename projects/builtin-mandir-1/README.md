# Built-in Mandir 1

A masterfully integrated spiritual sanctuary, custom-built to blend seamlessly with the home's architecture. This design features elegant cabinetry and integrated lighting to create a serene prayer space that maximizes floor space.

## Features
- **Integrated Design:** Custom-fitted into existing alcoves for a clean look.
- **Ambient Lighting:** Built-in LEDs to highlight the sacred murtis.
- **High-Quality Plywood:** Ensures durability and a premium finish.
- **Custom Hardware:** Minimalist handles that complement the clean lines.

## The Process
Crafted with precision using hand-selected birch plywood. The installation focused on achieving tight tolerances and a "designed-in" feel.
