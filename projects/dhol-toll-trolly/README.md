# Dhol Toll Trolly

A unique solution for mobile cultural performances. This custom trolly is designed to securely transport and display a Dhol, combining mobility with a stable performance platform.

## Features
- **Heavy-Duty Mobility:** Lockable casters for easy transport across various terrains.
- **Secure Instrument Mounts:** Designed to hold the Dhol firmly during movement.
- **Reinforced Frame:** Built to handle the weight and vibrations of the instrument.
- **Compact Storage:** Sized to fit through standard doorways.
